# Arithmetic-Game
A game for any age that enhances the solving arithmetic skill in 1-3 digits The game has a limited time of 1 minute. Solve as many as you can.

The game has 3 levels:

Easy: 1 digit

Moderate: 2 digits

Difficult: 3 digits

Here are the operations options:

Addition

Subraction

Multiplication

Division
